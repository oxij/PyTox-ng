About
-----

|project| is a set of object-oriented Python bindings for `c-toxcore <https://github.com/TokTok/c-toxcore>`_ of `Tox.chat <https://tox.chat/>`_ instant messaging system.

The project page/repository is located at `GitHub <https://github.com/oxij/PyTox-ng>`_.
